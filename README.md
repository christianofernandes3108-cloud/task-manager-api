# Task Manager API

A backend API built with Node.js, Express, and MongoDB.  
Supports user authentication and CRUD operations for tasks.

## How to install

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with the required environment variables
4. Run the server using `npm run dev` or `npm start`

## Project Structure

- controllers/
- middleware/
- models/
- routes/
- server.js

## Features

- User registration & login (JWT authentication)
- Protected routes
- Task creation, editing, and deletion
